ACCU-WEATHER

A simple Weather App

About

This is simple weather app that provides a 5-day weather forecast based on a given location.

Mapbox API is used for the map, map interactions, and geolocation.

Open Weather Map API is used to retrieve weather information when given a location.

Uses JavaScript to make AJAX calls to the Open Weather Map API and Mapbox API.

Animista a CSS animation generator is used for the animations of the app.

Boxicons is used to aquaire icon for the nav logo.
